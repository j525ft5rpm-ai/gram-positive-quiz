# 革兰阳性菌题库答题网站部署说明

这是一个静态网站，手机和电脑都可以访问。题库已经内置在 `index.html` 中，不依赖数据库，也不需要服务器程序。

## 方法一：Netlify 拖拽部署

1. 打开 Netlify 官网并登录。
2. 进入 Sites 页面，选择 Add new site 或直接拖拽部署。
3. 上传整个 `gram_positive_quiz_site` 文件夹，或上传 `gram_positive_quiz_site.zip`。
4. 部署完成后，Netlify 会给你一个网址，用手机打开即可使用。

## 方法二：GitHub Pages

1. 新建一个 GitHub 仓库。
2. 上传 `index.html`。
3. 在仓库 Settings → Pages 中启用 GitHub Pages。
4. 等待生成网站链接后，用手机访问即可。

## 方法三：局域网临时使用

如果只是想在同一个 Wi-Fi 下用手机临时访问，可以在电脑上运行：

```bash
python -m http.server 8000
```

然后在手机浏览器访问电脑的局域网 IP 加端口，例如：

```text
http://电脑IP:8000/
```

这种方式只适合临时使用，电脑关机或断网后就无法访问。
